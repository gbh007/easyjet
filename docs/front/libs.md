# Список используемых библиотек для TypeScript/Vue

## Основные зависимости

- **Vue 3** (`vue@^3.5.26`) — прогрессивный JavaScript-фреймворк для создания пользовательских интерфейсов
- **Vue Router 5** (`vue-router@^5.0.3`) — официальный роутер для Vue.js
- **Pinia** (`pinia@^3.0.4`) — официальное решение для управления состоянием приложения
- **Vuetify 4** (`vuetify@^4.0.0`) — UI-фреймворк с компонентами Material Design

## Утилиты

- **Axios** (`axios@^1.13.6`) — HTTP-клиент для выполнения запросов к backend API
- **@fontsource/roboto** (`@fontsource/roboto@^5.2.9`) — шрифт Roboto для Material Design
- **@mdi/font** (`@mdi/font@7.4.47`) — иконки Material Design Icons

## Инструменты разработки

- **TypeScript** (`typescript@~5.9.3`) — статическая типизация JavaScript
- **Vite** (`vite@^8.0.0`) — современный сборщик проектов
- **ESLint** (`eslint@^9.39.2`) — линтер кода
  - `eslint-config-vuetify@^4.3.4` — конфигурация для Vuetify
  - `eslint-config-prettier@^10.1.8` — отключение конфликтующих правил ESLint с Prettier
  - `@vue/eslint-config-prettier@^10.2.0` — конфигурация ESLint для Vue с Prettier
- **Prettier** (`prettier@^3.8.1`) — форматирование кода
- **vue-tsc** (`vue-tsc@^3.2.0`) — проверка типов TypeScript для Vue-компонентов

## Стилизация

- **Sass** (`sass-embedded@^1.97.1`) — CSS-препроцессор (Sass/SCSS)
- **unplugin-fonts** (`unplugin-fonts@^1.4.0`) — автоматическая загрузка шрифтов в Vite
- **vite-plugin-vuetify** (`vite-plugin-vuetify@^2.1.3`) — интеграция Vuetify с Vite
- **@vitejs/plugin-vue** (`@vitejs/plugin-vue@^6.0.5`) — плагин Vite для Vue 3

## Конфигурации

- **@tsconfig/node22** (`@tsconfig/node22@^22.0.5`) — TypeScript конфигурация для Node.js 22
- **@vue/tsconfig** (`@vue/tsconfig@^0.9.0`) — TypeScript конфигурация для Vue проектов
- **@types/node** (`@types/node@^22.0.5`) — типы для Node.js API
