# Список используемых библиотек для TypeScript/Vue

## Основные зависимости

- **Vue 3** — прогрессивный JavaScript-фреймворк
- **Vue Router 5** — официальный роутер для Vue
- **Pinia** — официальное решение для управления состоянием
- **Vuetify 4** — UI-фреймворк с компонентами Material Design

## Утилиты

- **Axios** — HTTP-клиент для запросов к API
- **@fontsource/roboto** — шрифт Roboto
- **@mdi/font** — иконки Material Design

## Инструменты разработки

- **TypeScript** — типизация JavaScript
- **Vite** — сборщик проектов
- **ESLint** — линтер кода
  - `eslint-config-vuetify` — конфиг для Vuetify
  - `eslint-config-prettier` — отключение конфликтующих правил ESLint
- **Prettier** — форматирование кода
- **vue-tsc** — проверка типов Vue

## Стилизация

- **Sass (sass-embedded)** — CSS-препроцессор
- **unplugin-fonts** — автозагрузка шрифтов
- **vite-plugin-vuetify** — интеграция Vuetify с Vite
